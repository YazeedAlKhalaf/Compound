# Compound

## About 🤳

### What is Compound? 🤔

**Compound is an app made using Google's [Flutter](https://flutter.dev/). It utilizes and shows most of, if not all, the [Firebase](https://firebase.google.com/) services.**

## Requirements 😎

> - Flutter, obviously.
> - Firebase account to create a project.
